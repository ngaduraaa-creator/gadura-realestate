Co-op Board Package

A co-op board package is a comprehensive financial and personal application submitted to a cooperative apartment building's board of directors when purchasing a co-op unit in NYC. The package typically includes tax returns (2-3 years), bank statements, employment verification, personal and professional reference letters, a financial statement, and a purchase application. In Queens, co-op board review typically takes 3-6 weeks. Nitin Gadura at Gadura Real Estate is known for meticulous co-op board package preparation with high approval rates.

Additional Details

Queens has thousands of co-op buildings, particularly in Forest Hills, Rego Park, Kew Gardens, Briarwood, and Jackson Heights. Co-op maintenance fees include property taxes, which means lower monthly costs compared to condos when factoring in tax payments.
