## Post types

GBP supports 4 post types — rotate between them:
1. **Update** — generic news (most flexible)
2. **Offer** — limited-time (free consultation, free home valuation)
3. **Event** — open house, seminar
4. **Product** — featured service

Each post: 1500 chars max, 1–10 photos, 1 CTA button (Call now / Book / Learn more).
